# H2 H1 후보군 분포 검정 결과

## 설계

- 입력: `data/features_monthly.csv`, `data/cluster_labels.csv`
- 보조 입력: `data/h1_current_candidates.csv`
- 기본 라벨: `is_stagnant_cluster` (H1 K-Means 성장 정체/주차 후보 cluster)
- 보조 라벨: `is_current_parking_candidate`, `is_high_confidence_candidate`
- 사전 정의 범주: 레벨 `270-279 / 280-285 / 286-290`, 직업 계열 `전사 / 마법사 / 궁수 / 도적 / 해적`
- 검정: 카이제곱 독립성 검정, `alpha = 0.05`; 효과크기 Cramer's V; 후보 셀 표준화 잔차
- 보수적 점검: 고정 주변합 Monte Carlo 검정(100,000회), 두 기본 교차표에 대한 Holm 보정

H1 클러스터 후보는 471명(23.56%)이다. 분석 표본은 1,999명이다.

## 판정

H1 클러스터 후보 471명 기준으로 레벨 구간 분포는 Holm 보정 후에도 유의하다(p=1.272e-08, Cramer's V=0.137). 후보는 `270-279` 구간에 집중되고(비율 30.12%) `286-290` 구간에서는 낮다(비율 15.97%). H2의 레벨 구간 불균형 가설을 지지한다.

직업 계열 분포는 Holm 보정 p=0.04797로 유의하나, Cramer's V=0.069로 효과크기는 무시할 수준이다(대표본 χ² 민감도). 따라서 H2 결론은 `레벨 구간 집중은 확인(작은~중간 효과), 직업 계열은 실질적 집중 없음(효과크기 무시 가능)`으로 정리한다. 이 결과는 H1 파생 후보 라벨의 분포 검정이며 실제 주차 유저 ground truth 검정은 아니다.
